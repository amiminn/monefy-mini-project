# Mini Project - Monefy

## Development

| Runner         |          Tech           |
| -------------- | :---------------------: |
| php            |         php 8.3         |
| database       |          mysql          |
| nodejs         |           bun           |
| backend        |       laravel 12        |
| frontend       | vite (react-typescript) |
| design pattern |        mvc - api        |

## Setup lalu running

```
composer install && bun install
```

```
php artisan serve && bun run dev
```
